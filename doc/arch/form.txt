Заметки про форму

1) Возможность отрендерить произвольную схему - это один компонент
2) Возможность отрендерить схему сущности + ссылки внутри схемы
3) Возможность рендерить вложенную схему
4) Возможность рендерить array of type + array of schema
5) Сохранение сущности по сабмиту формы
6) Возможность задания промежуточной модели, хуков всяких

Вложенная схема не может содержать ссылки
Прилинкованная сущность может содержать свои ссылки
Ссылки ренерятся под-формами
